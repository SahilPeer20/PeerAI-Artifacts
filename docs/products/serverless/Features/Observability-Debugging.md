# Feature: Observability & Debugging

**Product:** Serverless

**Epic:** s: Serverless Framework v4

**Priority:** Medium

**Status:** Generated

## Description

**As a** developer, SRE, or support engineer  
**I want** to monitor, debug, and resolve issues in my serverless apps without leaving my workflow  
**So that** I can quickly identify and fix problems, reducing MTTR and cloud costs

**Summary**:  
Observability & Debugging brings together real-time log streaming, remote and local function invocation, service info display, and integrations with the Serverless Dashboard and third-party tools like Axiom. It includes AI-assisted error summaries and automated support report generation, empowering developers and teams to maintain service health and respond quickly to incidents.
