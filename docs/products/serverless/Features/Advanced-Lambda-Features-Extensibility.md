# Feature: Advanced Lambda Features & Extensibility

**Product:** Serverless

**Epic:** s: Serverless Framework v4

**Priority:** Medium

**Status:** Generated

## Description

**As a** backend or AI/ML engineer  
**I want** to use the latest AWS Lambda features and easily extend my applications with plugins  
**So that** I can build modern, secure, and scalable serverless workloads for any use case

**Summary**:  
This feature combines advanced Lambda options (e.g., Managed Instances, Durable Functions, per-function IAM, response streaming), built-in support for popular plugins (e.g., Python requirements, AppSync, prune), and a robust plugin system. It empowers users to adopt new AWS features, enforce least-privilege security, and quickly add new capabilities as the ecosystem evolves.
