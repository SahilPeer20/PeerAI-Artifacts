# NFR: Data Encryption in Transit

**Product:** Serverless

**Category:** Security

**Priority:** Critical

## Description

All data transmitted between client applications and the serverless platform must be encrypted using industry-standard protocols such as TLS 1.2 or higher.

## Measurement Criteria

Verify that all network traffic is encrypted using TLS 1.2 or above via penetration testing and network analysis.

## Acceptance Threshold

100% of data in transit is encrypted using TLS 1.2 or higher.
